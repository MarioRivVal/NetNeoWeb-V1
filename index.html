Student Management

Code

This project was developed for a dedicated educator who entrusted me with the task of creating a robust and efficient solution. Following the Model-View-Controller (MVC) architecture I designed this application. It seamlessly utilizes IndexedDB, making it a fully-fledged Progressive Web App (PWA).

The primary objective of this application is to facilitate the management of student records, including detailed information and payment tracking. Periodically, every three months, it automatically identifies and generates bills for students who have made payments, streamlining the administrative process for the teacher.

  • Sass 1.70
  • Json
  • LocalStorage

TableOrder Pro

Code

This project involved the creation of a user-friendly application tailored for both laptops and mobile devices, specifically designed to streamline restaurant operations. The app allows restaurant staff to efficiently manage tables, mark them as 'busy,' and take customer orders from a predefined list of products sourced from JSON data.

Additional features include the ability to add tips, automatically calculate the final total, and display the current time for timely service. At the end of the dining experience, the app facilitates the generation of a detailed bill of consumptions for customers, enhancing the overall dining experience and simplifying restaurant management.

  • JavaScript ES12
  • Sass 1.70
  • API Call
  • LocalStorage

RecipeHub

Code

RecipeHub is a visually appealing and fully responsive application that combines style with functionality. Users can effortlessly explore a world of culinary delights by making API calls to a vast repository of recipes. The app offers a user-friendly interface to search for specific meal types, providing detailed recipe information and the option to save favorites.

To enhance the user experience, all data, including favorite recipes, is securely stored in local storage for future reference. With its delightful colors, smooth animations, and seamless responsiveness, RecipeHub is your gateway to culinary inspiration.
